 # Data Types:
 
 - Data Types are of two types.
    -   primitive
    -   non-primitive

    
### Primitive Data Type:
-   From the name it suggests these are the simplest data types:


```aidl

Type        default     size 

booelan     false       1 bit
byte        0           8 bits
char        \u0000      16 bits
short       0           16 bits
int         0           32 bits
long        0           64 bits
float       0.0         32 bits
double      0.0         64 bits
```
